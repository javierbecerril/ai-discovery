# Insight Registry

Living document. All insights from all inputs accumulate here.
Do not delete entries — merge duplicates using dedup-clusterer instead.

---

## How to Use

- **intake-synthesizer** adds new INSIGHT entries at the bottom.
- **dedup-clusterer** adds THEME entries and updates INSIGHT entries with theme tags and duplicate flags.
- Every insight must trace back to a source input.
